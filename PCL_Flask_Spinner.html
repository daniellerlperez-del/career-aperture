Usage notes

Drop the inline SVG anywhere you'd use a spinner or loading indicator. Scale by changing the width attribute — the viewBox stays fixed. Background is transparent so it works on any dark surface.

For light backgrounds, invert the stroke colors: use rgba(8,7,6,0.8) for the flask outline and rgba(8,7,6,0.6) for bubbles.

If using multiple spinners on the same page, give each gradient a unique id (e.g. sl-1, sl-2) to avoid conflicts.
