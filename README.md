# Career Aperture

**Built by a Principal PM who needed one place for everything — and built it instead of buying it.**

Where the [PCL website](https://perez-creative-labs.netlify.app) shows what gets built for clients — Career Aperture shows what gets built for the builder. Career work treated as craft.

This is what it looks like when a PM builds the tool herself.

## Live

→ [career-aperture.netlify.app](https://career-aperture.netlify.app)

## Modes

| Mode | Name | What it does |
|------|------|-------------|
| Shoot | Job Search | Precision URL builder for LinkedIn, The Ladders, Indeed, Glassdoor, BuiltIn, Wellfound |
| Develop | Kanban Board | Unified pipeline — Job Hunt + SMB Business Dev + Exposure Log |
| Loupe | Contact Intel | Claude-powered contact dossier + personalized outreach draft |
| Field Work | SMB CRM | SMB prospecting + PCL pitch generator + pipeline add |
| Reach | Networking | Personalized networking outreach message generator |
| Claims | Watch List | Separation timeline, deadlines, and recurring action tracker |
| KPI | Analytics | API cost, model distribution, success rate, environment links |

## Built With

- Single-file React (all inline — no bundler, no CDN dependency)
- Claude API via Netlify serverless function proxy
- HomeFlowOS for source-of-truth context (career, legal, business domains)
